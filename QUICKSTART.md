# ⚡ Quick Start

## Em 1 Comando

```bash
docker-compose up
```

Pronto! ✅

---

## URLs dos Serviços

| Serviço | URL |
|---------|-----|
| 🎨 Frontend | http://localhost:3000 |
| 🔌 Backend API | http://localhost:8000 |
| 📖 API Docs | http://localhost:8000/docs |
| 🗄️ Database | localhost:5432 (user/password) |

---

## Parar os Serviços

Pressiona `CTRL + C` no terminal

---

## Troubleshooting

### Port already in use
```bash
docker-compose down
# Depois tenta de novo
docker-compose up
```

### Docker not found
→ Instala Docker Desktop: https://www.docker.com/products/docker-desktop
